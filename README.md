##
 ### Installation
  ```
  # clone the repository
  $ git clone https://github.com/SamueleAmato/Lawliet
  
  # change the working directory to sherlock
  $ cd Lawliet

  # install the requirements
  $ pip3 install -r requirements.txt
  ```
##
### Usage
  ```
  //Linux
  bash lawliet.sh
  
  //Windows
  lawliet.exe
  ```
